Smart Budget Tracker Site Plan

WDD 231 Individual Website Project

Site Name

Smart Budget Tracker

This name was selected because it clearly describes the purpose of the website, which is to help users manage their income, expenses, and savings in a smart and organized way.

Optional Domain: smartbudgettracker.com

Site Purpose

The Smart Budget Tracker website provides a simple platform for users to record income and expenses, monitor spending habits, view financial summaries, and maintain better control over personal finances.

Scenarios

  • How much money have I spent this month?
  • What is my current balance after recording all expenses?
  • Which category is taking the largest portion of my budget?
  • Can I view my recent transactions in one place?
